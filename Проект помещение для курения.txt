1.2. Опрос педагогов на тему "Необходимость наличия помещения для курения около университета"
Был проведен опрос среди педагогов
Результат этого опроса - помещение для курения имеет место быть около университета для пользования студентами и сотрудниками Череповецкого государственного университета, однако, это помещение 
не должно находиться под окнами кабинета кафедры Математического и программного обеспечения ЭВМ, чтобы табачный дым не попадал внутрь кабинета через открытые окна. 


2.1  Планирование размещения
Общая схема представлена на рис. N. Помещение для курения имеет 
закругленную крышу из поликарбоната, три железобетонные стены (четвертая
является входом в помещение), две скамьи напротив друг друга.

Планирование сметы
Затраты на необходимые материалы для создания помещения для курения обуславливаются
тем, что здание должно быть крепкое, конструкция надежная и долговечная. Грамотный 
подход к выделению бюджетных средств для стройматериалов способствует соблюдению
безопасности для жизни студентов и сотрудников Череповецкого государственного 
университета, находящихся в помещении для курения, таблица бюджета для создания
"помещения для курения" представлена в табл. 1

 Проект
не имеет финансирования на момент заполнения паспорта.
